# Architecture Decision Records

This directory contains Architecture Decision Records (ADRs). The format
follows the template in ADR-0001 (Status / Context / Decision / Consequences).

## Status vocabulary

| Status     | Meaning                                         |
|------------|-------------------------------------------------|
| Proposed   | Draft; decision not yet finalized               |
| Accepted   | In force                                        |
| Deprecated | No longer in force; replacement not yet decided |
| Superseded | Replaced by a later ADR (link provided)         |
