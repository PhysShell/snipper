# Privacy Policy

**Mask by default.** Unless explicitly configured otherwise, the project
collects no personal data and emits no telemetry.

## Data collected

| Category | Default | Notes |

| File paths | never stored | used transiently |
| Telemetry | disabled | opt-in only |
